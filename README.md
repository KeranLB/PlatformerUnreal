# PlatformerUnreal

Touches :
  - Z/Q/S/D -> se déplacer
  - Souris -> regarder
  - Espace -> sauter
  - Maintenir Espace -> planer
  - Shift Gauche -> dash

A Savoir :
  - Porte Bleu : besoin de trouver son orbe d'activation pour ouvrir / sert a se tp au niveau suivant.
  - Porte Rouge : Piège.
  - Porte Verte : Heal.

Expérience de jeu :
  "Die and retry", Réussis à passer tout les obstacles avec un nombre de vie limité ou recommence à 0.

Analyse de production :
  - Difficultés :
        La mécanique des anneaux de vents à était un peu complexe car les impulsions n'était pas les mêmes quand on le prenait en planant ou sans planer.

  - Mon travail :
        Je pense que mon travail à était plutôt productif dans le sens où j'ai bien compris tout les éléments utiliser que je ne connaissais pas avt, comme la TimeLine ou les BP_Component qui peuvent être ajouter à un Actor et globalement le BP ou je commence à être un peu plus à l'aise dans la façon de faire du BP.

  - j'aurais aimé avoir un " aire controlle", lors du planage, plus conséquent car il est jouable mais pas satisfaisant comme j'aurais aimé.
